## Getting Started

To install all Wally dependencies, run:

```bash
sh scripts/install.sh
```

To build the place from scratch, use:

```bash
rojo build -o "lua.rbxlx"
```

Next, open `lua.rbxlx` in Roblox Studio and start the Rojo server:

```bash
rojo serve
```
